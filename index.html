    Password Manager

    Google passwords are great and all, but there isn’t any functionality for desktop applications. So, I thought, why not make my own password manager? So that’s what I did. I used Java with a swing interface and used AES and SHA-256 for secure password encryption and decryption.

    • Password Encryption Process:
      • Combination with Salt: The plaintext password is mixed with a randomly generated salt.
      • Hashing with SHA-256: The combined data undergoes SHA-256 hashing.
      • Encryption with AES: AES encryption, utilizing the master password as the secret key, is applied.
      • Storage in Passwords File: The encrypted password is stored in a designated file.
